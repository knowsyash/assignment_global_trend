## Task Manager

### Stack
- Backend: Node.js + Express (in-memory storage)
- Frontend: React + Vite

### How to run

**Backend:**
cd backend
npm install
node server.js

**Frontend (in a new terminal):**
cd frontend
npm install
npm run dev

Open http://localhost:5173

### Assumptions
- Tasks are stored in memory and reset on server restart
- No database or authentication required
